==========================================
    LINUX / TERMUX - ბრძანებების შპარგალკა
==========================================

1. pwd   - სად ვარ? (გაჩვენებს მისამართს)
2. ls    - რა არის აქ? (ფაილების სია)
3. cd    - გადასვლა (მაგ: cd Documents)
4. mkdir - საქაღალდის შექმნა
5. pkg   - პროგრამების მართვა (install/update)
6. cp    - კოპირება (copy)
7. mv    - გადატანა ან სახელის შეცვლა
8. rm    - წაშლა (სამუდამოდ!)
9. clear - ეკრანის გასუფთავება
10. exit - ტერმინალის დახურვა

------------------------------------------
მნიშვნელოვანი სიმბოლოები:
------------------------------------------
..  -> ერთი საფეხურით უკან დაბრუნება (cd ..)
~   -> მთავარი (Home) საქაღალდე
/   -> სისტემური ძირი (Root)
TAB -> ბრძანების ავტომატურად დასრულება
==========================================
