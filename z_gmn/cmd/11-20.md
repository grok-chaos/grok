==================================================
        LINUX ბრძანებების სრული გზამკვლევი
==================================================

1. pwd (Print Working Directory)
   - მნიშვნელობა: სამუშაო საქაღალდის ჩვენება.
   - მაგალითი: pwd
   (გეტყვის ზუსტად სად ხარ, მაგ:
 /data/data/com.termux/files/home)

2. ls (List)
   - მნიშვნელობა: ფაილების ჩამონათვალი.
   - მაგალითი 1: ls -l (დეტალური სია: ზომა, თარიღი)
   - მაგალითი 2: ls -a (აჩვენებს დამალულ ფაილებსაც)

3. cd (Change Directory)
   - მნიშვნელობა: საქაღალდეებს შორის გადაადგილება.
   - მაგალითი 1: cd Documents (შესვლა საქაღალდეში)
   - მაგალითი 2: cd .. (ერთი ნაბიჯით უკან დაბრუნება)

4. mkdir (Make Directory)
   - მნიშვნელობა: ახალი საქაღალდის შექმნა.
   - მაგალითი: mkdir vswavlob_linuxs
   - შესაძლებლობა: mkdir -p saxli/otaxi (შექმნის
 საქაღალდეს საქაღალდეში)

5. pkg (Package Manager)
   - მნიშვნელობა: პროგრამების მართვა.
   - მაგალითი 1: pkg install python (პითონის დაყენება)
   - მაგალითი 2: pkg search git (პროგრამის მოძებნა)

6. cp (Copy)
   - მნიშვნელობა: კოპირება.
   - მაგალითი 1: cp faili.txt asli.txt (ფაილის ასლი)
   - მაგალითი 2: cp -r papka1 papka2 (მთლიანი საქაღალდის
 კოპირება)

7. mv (Move)
   - მნიშვნელობა: გადატანა ან სახელის შეცვლა.
   - მაგალითი 1: mv dzveli.txt axali.txt (სახელის შეცვლა)
   - მაგალითი 2: mv faili.txt Documents/ (ფაილის გადატანა)

8. rm (Remove)
   - მნიშვნელობა: წაშლა.
   - მაგალითი 1: rm faili.txt (ფაილის წაშლა)
   - მაგალითი 2: rm -rf papka (საქაღალდის წაშლა მთელი
 შიგთავსით)

9. cat (Concatenate)
   - მნიშვნელობა: ფაილის შიგთავსის ჩვენება ეკრანზე.
   - მაგალითი: cat daxmareba.txt (სწრაფი წაკითხვა)

10. chmod (Change Mode)
    - მნიშვნელობა: ფაილზე უფლებების მინიჭება.
    - მაგალითი: chmod +x skripti.sh (ფაილის "გაშვებადად"
 ქცევა)

==================================================
სასარგებლო კლავიშები:
- CTRL + C: მიმდინარე პროცესის შეწყვეტა (გაჩერება)
- CTRL + L: იგივეა რაც 'clear' (ეკრანის გასუფთავება)
- ისარი ზევით: წინა დაწერილი ბრძანების დაბრუნება
==================================================


Linux-ში ბრძანებების უმეტესობას აქვს ე.წ. Flag-ები (ტირეით დაწყებული ასოები, მაგ: -l, -a, -r).
​-r ხშირად ნიშნავს Recursive (ანუ ეხება საქაღალდეს და მის შიგნით ყველაფერს).
​-h ხშირად ნიშნავს Human-readable (მაგალითად, ფაილის ზომას გიჩვენებს მეგაბაიტებში და არა ბაიტებში).
