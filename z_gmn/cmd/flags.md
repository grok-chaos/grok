==================================================
   FLAG-рЃћрЃЉрЃўрЃА (рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА) рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃљ
==================================================

1. -l (Long listing)
   - рЃарЃљрЃА рЃњрЃЋрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА: рЃцрЃљрЃўрЃџрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃА, рЃърЃљрЃбрЃарЃЮрЃюрЃА, 
     рЃќрЃЮрЃЏрЃљрЃА рЃЊрЃљ рЃЉрЃЮрЃџрЃЮ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃарЃЮрЃфрЃљ рЃњрЃўрЃюрЃЊрЃљ рЃњрЃљрЃўрЃњрЃЮ, рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃЊрЃўрЃЊрЃўрЃљ 
     рЃцрЃљрЃўрЃџрЃў рЃљрЃю рЃњрЃљрЃЦрЃЋрЃА рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃўрЃАрЃў рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљ.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: ls -l

2. -a (All / Hidden)
   - рЃарЃљрЃА рЃњрЃЋрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА: рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃА, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА 
     рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃЌ (.) рЃЊрЃљрЃгрЃДрЃћрЃЉрЃБрЃџ рЃЊрЃљрЃЏрЃљрЃџрЃБрЃџ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: Linux-рЃерЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў 
     (рЃЏрЃљрЃњ: .bashrc) рЃЊрЃљрЃЏрЃљрЃџрЃБрЃџрЃўрЃљ рЃЊрЃљ рЃЏрЃљрЃЌ рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ 
     рЃћрЃА Flag рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: ls -a

3. -r / -R (Recursive)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃА рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ 
     рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃќрЃћ, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃЏрЃўрЃА рЃЏрЃЌрЃћрЃџ рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃќрЃћ.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ (cp -r) рЃљрЃю 
     рЃгрЃљрЃерЃџрЃљ (rm -r). рЃљрЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ 
     рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃА "рЃЋрЃћрЃа рЃћрЃ«рЃћрЃЉрЃљ".
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: rm -r my_folder

4. -h (Human Readable)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃфрЃўрЃцрЃарЃћрЃЉрЃА (рЃЉрЃљрЃўрЃбрЃћрЃЉрЃА) рЃљрЃЦрЃфрЃћрЃЋрЃА 
     рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃцрЃЮрЃарЃЏрЃљрЃбрЃљрЃЊ (K, M, G).
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: ls -l-рЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ. рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ 1048576-рЃўрЃАрЃљ, 
     рЃЊрЃљрЃњрЃўрЃгрЃћрЃарЃА 1M.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: ls -lh

5. -f (Force)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃўрЃФрЃБрЃџрЃћрЃЉрЃўрЃЌрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ. 
     рЃБрЃњрЃБрЃџрЃћрЃЉрЃћрЃџрЃДрЃЮрЃцрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃА.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃарЃЮрЃфрЃљ рЃЉрЃћрЃЋрЃарЃў рЃцрЃљрЃўрЃџрЃўрЃљ рЃгрЃљрЃАрЃљрЃерЃџрЃћрЃџрЃў рЃЊрЃљ 
     рЃљрЃа рЃњрЃўрЃюрЃЊрЃљ рЃАрЃљрЃЌрЃўрЃЌрЃљрЃЮрЃЊ рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃЮ.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: rm -rf folder_name

6. -i (Interactive)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: -f-рЃўрЃА рЃАрЃљрЃърЃўрЃарЃўрЃАрЃърЃўрЃарЃЮрЃљ. рЃњрЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ 
     рЃЊрЃљрЃАрЃбрЃБрЃарЃА рЃДрЃЮрЃЋрЃћрЃџ рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃќрЃћ.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃарЃЮрЃфрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю рЃцрЃљрЃўрЃџрЃћрЃЉрЃА 
     рЃерЃџрЃў рЃЊрЃљ рЃњрЃўрЃюрЃЊрЃљ рЃЊрЃљрЃќрЃдрЃЋрЃћрЃБрЃџрЃў рЃўрЃДрЃЮ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃАрЃњрЃљрЃю.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: rm -i file.txt

7. -v (Verbose)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃљрЃА.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃАрЃљрЃА, 
     рЃарЃЮрЃЏ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮ, рЃарЃљ рЃћрЃбрЃљрЃърЃќрЃћрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃў.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: cp -v file1.txt storage/

8. -p (Parents)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃЦрЃЏрЃюрЃўрЃА рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃа рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃћрЃЉрЃАрЃљрЃф.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃњрЃўрЃюрЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљ рЃњрЃќрЃљ a/b/c, рЃЏрЃљрЃњрЃарЃљрЃЏ 
     рЃљрЃарЃф 'a' рЃњрЃљрЃЦрЃЋрЃА рЃЊрЃљ рЃљрЃарЃф 'b'. рЃћрЃА Flag 
     рЃАрЃљрЃЏрЃўрЃЋрЃћрЃА рЃћрЃарЃЌрЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: mkdir -p work/2024/january

9. -u (Update)
   - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА/рЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ 
     рЃцрЃљрЃўрЃџрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ.
   - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљ.
   - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: cp -u source.txt destination.txt

10. -y (Assume Yes)
    - рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА: рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА "Yes"-рЃА.
    - рЃАрЃфрЃћрЃюрЃљрЃарЃў: рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃАрЃљрЃА, рЃарЃЮрЃЏ 
      рЃљрЃа рЃЏрЃЮрЃњрЃўрЃгрЃўрЃЮрЃА рЃћрЃЎрЃарЃљрЃюрЃЌрЃљрЃю рЃџрЃЮрЃЊрЃўрЃюрЃў.
    - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ: pkg upgrade -y

­ЪњА рЃЊрЃљрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћ: Linux-рЃерЃў Flag-рЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ 
рЃБрЃцрЃарЃЮ рЃФрЃџрЃўрЃћрЃарЃўрЃљ: 'ls -alt' рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃА 
рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ, рЃЊрЃарЃЮрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃА.

Flag-рЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃўрЃарЃћрЃЉрЃљ
РђІLinux-рЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ Flag-рЃћрЃЉрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА рЃњрЃљрЃЊрЃљрЃљрЃЉрЃљ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ,
 рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃЊрЃљрЃгрЃћрЃарЃЮ ls -l -a -h, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃгрЃћрЃарЃЮ:
ls -lah Рђћ рЃћрЃА рЃћрЃарЃЌрЃЉрЃљрЃерЃљрЃЊ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА: рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃАрЃўрЃљрЃА (l), рЃЊрЃљрЃЏрЃљрЃџрЃБрЃџ
 рЃцрЃљрЃўрЃџрЃћрЃЉрЃА (a) рЃЊрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃќрЃЮрЃЏрЃћрЃЉрЃА (h).
РђІРџа№ИЈ рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ: rm -rf
РђІрЃћрЃА рЃљрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃЊрЃљ рЃАрЃљрЃерЃўрЃерЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ Linux-рЃерЃў.
РђІ-r (рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃћрЃЉрЃўрЃА рЃгрЃљрЃерЃџрЃљ) + -f (рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃгрЃљрЃерЃџрЃљ).
рЃЌрЃБ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃљрЃарЃљрЃАрЃгрЃЮрЃа рЃљрЃЊрЃњрЃўрЃџрЃљрЃА рЃЊрЃљрЃгрЃћрЃа rm -rf *, рЃўрЃА рЃгрЃљрЃерЃџрЃўрЃА
