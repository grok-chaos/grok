# Git-ის დაყენება და გამოყენება Termux-ში

**Install:** pkg install git

**Usage:** git clone [url] — რეპოს კლონვა  
git add . — ფაილების დამატება  
git commit -m "მესიჯი" — ცვლილებების დაფიქსირება  
git push — ატვირთვა GitHub-ზე

**რისთვის გამოიყენება:** ვერსიების კონტროლი, ნოუთების უსაფრთხო შენახვა ♥️
